## 安装

```
$ git clone https://github.com/luckyjing/package <your_project_name>

// 安装依赖
$ npm install

// 执行test目录下的所有测试用例，并且呈现测试覆盖率报告
$ npm run build

// 模块ES6转换为ES5
$ npm run build

// [todo] 产生公共资源（如*.min.js）
$ npm run dist

// 发布

```

## 功能

- 支持`Babel`，所有的ES6语法模块转换为ES5模块后进行发布
